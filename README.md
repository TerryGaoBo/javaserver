# javaserver
基于java语言实现的游戏服务器框架

* HTTP 服务器
* SOCKET 服务器

可作为移动游戏，网页游戏，app等服务器架构

##########################################################

* netty
* spring
* json
* mybatis
* resteasy
* servlet
* commons
* log4j
* mysql-connector-java
* quartz

#数据存储方式：
